mlp_smt_closed.mlp.plot3d.plot_dist_map(x_test, y_test, y_prediction, name)¶

Plot the distance between pairs of 2D points on a 2D grid in 3D.

This module can be used to plot the (2 norm) distance between two points \(y, y' \in \mathbb{R}^2\) for a given grid (\(\subset \mathbb{R}^2\)) of pairs. Its aim is to visualize the prediction error of an MLP trained on a function \(f \in \mathbb{R}^2 \rightarrow \mathbb{R}^2\) with input \(x \in \mathbb{R}^2\) and output \(y' \in \mathbb{R}^2\).

Parameters

  • x_test – (n,2)-array; list of all points x_test[i] \(\in \mathbb{R}^2\) in the grid. (inputs for the MLP)

  • y_test – (n,2)-array; list of all points y_test[i] = f(x_test[i]) \(\in \mathbb{R}^2\), where f is the function on which the MLP was trained

  • y_prediction – (n,2)-array; list of all points y_prediction[i] \(\in \mathbb{R}^2\), where y_prediction[i] is the prediction of the MLP trained on ``f for input x_test[i]

  • name – name of the function f on which the MLP was trained, used to name the file in which the visualization is saved.
