training module¶

This module can be used to create an MLP, train it a function f, save it, and plot the error. So far, f \(\in \mathbb{R} \rightarrow \mathbb{R}\) or f \(\in \mathbb{R}^2 \rightarrow \mathbb{R}^2\) is supported

mlp_smt_closed.mlp.training.open_model(func_class)¶

Print the saved model/parameters of an MLP trained on func_class if the model was trained on func_class already

Parameters

func_class – function for which the model should be printed

mlp_smt_closed.mlp.training.train1d(func_class)¶

Create an MLP, train it on a function f \(\in \mathbb{R} \rightarrow \mathbb{R}\) contained in func_class, save it, and plot the error

Parameters

func_class – a class which must contain a static function f \(\in \mathbb{R} \rightarrow \mathbb{R}\) and a name attribute

mlp_smt_closed.mlp.training.train2d(func_class)¶

Create an MLP, train it on a function f \(\in \mathbb{R}^2 \rightarrow \mathbb{R}^2\) contained in func_class, save it, and plot the error

Parameters

func_class – a class which must contain a static function f \(\in \mathbb{R}^2 \rightarrow \mathbb{R}^2\) and a name attribute
