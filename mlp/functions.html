functions module¶

This module contains classes for functions which the MLP can learn.

class mlp_smt_closed.mlp.functions.FuncClass¶

Bases: abc.ABC

Abstract parent class for all functions

abstract property degree¶

This function returns the degree of the function

abstract property dimension¶

This function returns the dimensionality of the function. Note that at this point only a very small portion of multidimensional functions are supported, namely all of the form \(f: x \in \mathbb{R}^n \mapsto a\cdot x+b\) with \(a \in \mathbb{R}^{n \times n}, b \in \mathbb{R}^n\)

abstract f(x)¶

This function returns the value of the \(f\) for input \(x\)

abstract property name¶

This function returns the name of the function.
