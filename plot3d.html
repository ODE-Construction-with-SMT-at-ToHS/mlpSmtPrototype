plot3d module¶

This module can be used to plot the (2 norm) distance between two 2D points for a given grid. Its aim is to visualize the prediction error of an MLP with 2D input and 2D output.

plot3d.plot_dist_map(x_test, y_test, y_prediction, name)¶

Plot the distance (2 norm) between y_test[i] and y_prediction[i] for all points x_test[i] on the grid x_test
